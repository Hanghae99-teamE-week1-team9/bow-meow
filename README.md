# Bow-Meow

잃어버린 반려 동물의 정보를 제보/등록하여 공유하는 서비스로서, 소중한 반려동물이 가정의 품으로 돌아갈 수 있도록 돕기 위해 만들었습니다.

## 1. 제작 기간 / 팀원 소개

- 2022년 11월 14일 ~ 2022년 11월 17일
- E반 9조
  - 김한욱: 로그인/로그아웃
  - 박수빈: 회원가입, 회원정보 수정 및 탈퇴
  - 박태근: 댓글 리스트 조회, 댓글 등록, 댓글 단일 조회
  - 전유진: 게시글 리스트 조회, 게시글 등록, 게시글 단일 조회
